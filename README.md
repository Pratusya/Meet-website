# Gujarat Profil Website

A professional and responsive website for Gujarat Profil - Precision Profile Cutting Solutions.

## File Structure

```
e:\Meet website\
├── index.html          # Main HTML file
├── styles.css          # All CSS styles and responsive design
├── script.js           # JavaScript functionality
├── logo.jpg           # Company logo image
├── CNC.jpg            # CNC machine image for hero section
└── README.md          # Project documentation
```

## Features

- **Fully Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Mobile Navigation**: Hamburger menu with smooth animations for mobile devices
- **Modern UI/UX**: Professional design with gradient backgrounds, animations, and hover effects
- **Optimized Performance**: Separated CSS and JavaScript for better loading times
- **Cross-browser Compatibility**: Works on all modern browsers
- **SEO Optimized**: Proper HTML structure and meta tags

## Sections

1. **Hero Section**: Eye-catching introduction with CNC machine image
2. **About Section**: Company information and statistics (image removed as requested)
3. **Services Section**: Six service cards with icons and descriptions
4. **Features Section**: Four feature cards highlighting capabilities
5. **Contact Section**: Contact information and workshop showcase
6. **Footer**: Links, social media, and company details

## Mobile Features

- **Hamburger Menu**: 3-bar animated button for mobile navigation
- **Touch-friendly**: All buttons and links optimized for touch interaction
- **Responsive Images**: Images scale properly on all screen sizes
- **Mobile-first CSS**: Optimized for mobile performance

## Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)

## Setup Instructions

1. Ensure all files are in the same directory
2. Open `index.html` in a web browser
3. All external dependencies (fonts) are loaded from CDN

## Images Required

- `logo.jpg`: Company logo (used in navigation and footer)
- `CNC.jpg`: CNC machine image (used in hero section)

## Performance Features

- **Lazy Loading**: Images load efficiently
- **CSS Optimization**: Minified and organized styles
- **JavaScript Optimization**: Clean, efficient code
- **Error Handling**: Graceful fallbacks for missing images
